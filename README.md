# 🎓 Campus Placement Recruitment System

A web-based application designed to streamline and automate the campus placement and recruitment process for students, companies, and placement officers.

---

## 📌 About the Project

The Campus Placement Recruitment System is developed to digitalize the placement cell operations of educational institutions. It eliminates manual paperwork and provides a centralized platform for managing student profiles, company registrations, job postings, and placement records.

---

## ✨ Features

- 🔐 **User Authentication** – Separate login for Students, Companies, and Admin
- 👨‍🎓 **Student Module** – Register, update profile, view job listings, apply for drives
- 🏢 **Company Module** – Post job openings, view student profiles, shortlist candidates
- 🛠️ **Admin Module** – Manage students, companies, drives, and placement records
- 📋 **Placement Drive Management** – Schedule and manage recruitment drives
- 📊 **Reports & Statistics** – Track placement status and generate reports

## 🚀 How to Run the Project

1. Clone the repository
```bash
   git clone https://github.com/Prasadchandrut/CAMPUS-PLACEMENT-RECRUITMENT-SYSTEM.git
```
2. Import the database file (`.sql`) into MySQL using phpMyAdmin
3. Place the project folder in your server's root directory (e.g., `htdocs` for XAMPP)
4. Start Apache and MySQL from XAMPP Control Panel
5. Open your browser and go to:http://localhost/placement/login.php
6. to get into admin page use(admin@gmail.com pw:admin)
